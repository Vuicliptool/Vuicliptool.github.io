Hướng dẫn cài đặt Vuiclip-Tool

1 Cài đặt Termux

Tải Termux từ F-Droid (không dùng Google Play Store):
Tải Termux từ F-Droid

2 Cập nhật hệ thống

Chạy lệnh sau trong Termux:

pkg update && pkg install python curl -y
3 Cài đặt thư viện Python

Chạy từng lệnh sau:

pip install requests
pip install beautifulsoup4
pip install colorama
pip install websocket-client
pip install pysocks
pip install rich
4 Tải Vuiclip-Tool

Tải file tool về máy:

curl -o AresTool.py https://raw.githubusercontent.com/Ares-TOOL/main/refs/heads/main/main.py
5 Chạy tool

Sau khi tải xong, chạy lệnh:

python AresTool.py

Nếu mọi thứ đúng, tool sẽ bắt đầu chạy!

Xử lý lỗi thường gặp

Lỗi 1: command not found: curl

Lỗi này xảy ra khi 'curl' chưa được cài đặt.

Cách khắc phục:

pkg install curl
Lỗi 2: No module named 'requests'

Lỗi này xảy ra khi thư viện chưa được cài đặt.

Cách khắc phục: Chạy lại lệnh cài thư viện ở bước 3

Lỗi 3: Permission denied

Lỗi này xảy ra khi file không có quyền thực thi.

Cách khắc phục:

chmod +x AresTool.py
